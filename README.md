# 基于food-101数据库的食物识别-大作业
## 使用方法
### 运行pip -r requirements.txt 以及bat文件
### 在终端运行`python train.py`训练模型
### 在终端运行`python test.py`使用自建数据库，需要命名为test_images，文件夹命名和json文件对应<img width="533" height="192" alt="image" src="https://github.com/user-attachments/assets/068840bb-0144-4d75-b032-7f2e010c357f" />
### 数据库food-101下载完之后，建立data/food-101，再运行程序，由于数据库太大，上传很慢，这里提供下载链接https://data.vision.ee.ethz.ch/cvl/datasets_extra/food-101/
